pitchtypes.utils.fifths_from_diatonic_pitch_class

pitchtypes.utils.fifths_from_diatonic_pitch_class(pitch_class)

Return the number of steps along the line of fifths corresponding to a diatonic pitch class.

Parameters:

pitch_class – a diatonic pitch class; character in A, B, C, D, E, F, G

Returns:

fifth steps; an integer in -1, 0, … 5
